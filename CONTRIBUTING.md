# 🤝 Contributing Guidelines

Thank you for your interest in contributing! Please follow the steps below to submit your project.

---

## 📌 How to Submit

1. Fork this repository (`project-submet`)
2. Clone your fork
3. Create a new folder with your project name
4. Add your project files inside the folder
5. Include the required documentation
6. Commit your changes
7. Push to your fork
8. Create a Pull Request (PR)

---

## 📄 Documentation Requirement

Each submission must include a **3-page documentation** explaining:

- 🎯 Project Objectives  
- ⚙️ Implementation Details  
- 📊 Key Outcomes  

Accepted formats:
- README.md  
- PDF  
- Word document
